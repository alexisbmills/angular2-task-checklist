# angular2 task demo

## Install
### Clone repository
```
git clone git@bitbucket.org:alexisbmills/ng2-task.git
```
### Install dependencies
```
npm install -g node-sass live-server 
npm install
```

### Virtual host
See `apache-virtualhost.dist` 

## Run
Run typescript converter and launch project
```
npm run tsc
npm start
```
